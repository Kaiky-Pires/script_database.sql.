# script_database.sql.

​# Projeto Lógico de Banco de Dados - E-commerce
​Este repositório contém o desafio de projeto da DIO que visa a modelagem e criação de um banco de dados para um cenário de E-commerce.
​
# O que foi implementado:
​Modelo Lógico: Refinamento do modelo com distinção entre Clientes PF (Pessoa Física) e PJ (Pessoa Jurídica).
​Gestão de Entregas: Adicionado status de entrega e código de rastreio para melhor controlo logístico.
​Formas de Pagamento: Possibilidade de registar múltiplas formas de pagamento por cliente.
​Scripts SQL:
​Criação da estrutura das tabelas (Schema).
​Inserção de dados de teste (Persistência).
​Queries SQL complexas para análise de dados.
​
#Perguntas respondidas pelas Queries:
​Quantos pedidos foram feitos por cada cliente?
​Relação de nomes dos fornecedores e nomes dos produtos.
​Lista de clientes que gastaram acima de um determinado valor (usando HAVING).
